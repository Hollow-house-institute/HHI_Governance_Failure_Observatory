# Governance Failure Taxonomy

| Failure Domain | Description | Runtime Risk |
|---|---|---|
| Anthropomorphic Drift | Human-AI relational escalation and authority confusion | Boundary degradation |
| Recursive Amplification | Internal feedback loop escalation across time | Drift acceleration |
| Governance Continuity Failure | Evidence/provenance instability | Replay loss |
| Multi-Agent Oversight Failure | Coordination and escalation ambiguity | Accountability diffusion |
| Local AI Governance Loss | Telemetry visibility degradation at edge/runtime | Oversight discontinuity |
| Behavioral Drift | Runtime behavioral deviation accumulation | Governance instability |
